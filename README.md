# Discord Voice Bot

This Is A Simple Discord Bot Script Created With discord.js

The Only Thing This Bot Can Do Is It Can Join Your Voice Channel And It Play Nothing

You Can Use This Bot To Increase Your Discord Voice Count

# Run The Bot

#1

You Need To Install node.js To Run The Bot You Can Download It From The Link Bellow 

https://nodejs.org/en/

#2

After Installing Node You Should Get Your Bot Token In The Bot Script

If You Already Have A Bot Token Go To bot.js Edit It Go To Line 31 Replace "YOUR-TOKEN-HERE" With Your Bot Token Like This client.login("NzMwNzQyNjQ4NjA3OTk4MDI0.Xwb7Hw.GMC6KSxZbrAbpWyq94jg7Vc1uzI")

If You Dont Had One Go To Discord Developer Portal And Make A Bot Then Get A Token

https://discord.com/developers/applications

Be Sure To Save Your Token In bot.js

#3

Run The File Called run.bat And The Bot Will Run In Your Computer And If You Want The Bot To Be Turned Off Simply Close The Command Prompt

If The Bot Runned Correctly You Should See The Bot Username In Command Prompt

#4 

To Let The Bot Join Your Voice Channel Go To A Voice Channel In A Server Which The Bot Is In The Server

Then Go To A Text Channel And Type /join Or /voice  And The Bot Will Join Your Voice If It Has Permission To Read Messages In The Text Channel You Typed /join or /voice And Has Permission To Connect In The Voice Channel You Are Connected
